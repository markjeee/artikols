# README

This is a sample Rails application, to demonstrate a few API calls for
Owners and Articles.

# Development

```
git clone http://github.com/markjeee/artikols.git
cd artikols
bundle
rails db:setup
rails s
```

Then visit: http://127.0.0.1:3000
